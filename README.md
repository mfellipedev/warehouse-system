# Warehouse System

Um sistema de gerenciamento de estoque desenvolvido em Java puro, capaz de suportar até 26 ruas organizadas em ordem alfabética (A–Z). O sistema oferece as operações de armazenamento, transferência entre endereços, remoção e consulta de produtos.

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java 25
- **Testes:** JUnit 5
- **Build:** Maven

---

## ✅ Requisitos para Execução

- Java 25+
- IDE compatível com Java (IntelliJ IDEA recomendado)
